ILLUSTRATING AXIS IN FLEXBOX

the blue line which is the vertical lines represent the cross axis whereas the red line horizontal line, represents the primary axis. it is relevant to notice this is designed to be flex-direction:row;
The children will be positioned by default according to the following 2 rules:
Primary axis: Children will be bunched up at the start of the container.
Cross axis: Children will stretch out to fill the entire container.

Flexbox is all about controlling the distribution of elements in a row or column
we can flip to a column with the flex-direction property
With flex-direction: row, the primary axis runs horizontally, from left to right. When we flip to flex-direction: column, the primary axis runs vertically, from top to bottom.
